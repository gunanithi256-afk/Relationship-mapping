Introduction

Automating data import and mapping relationships is essential for efficient data handling. This approach reduces manual work and ensures data accuracy across systems.

What are Import Sets?

Import Sets act as staging tables that temporarily store incoming data before it is transformed into target tables. They help clean, validate, and process raw data efficiently.

What is Dot Walking?

Dot Walking allows you to access fields from related tables by referencing them using dot notation. It simplifies retrieving linked data without complex queries.

user.department.manager.name

Automation Workflow

  1. Load data into Import Set
  2. Create Transform Map
  3. Map source to target fields
  4. Use Dot Walking for relationships
  5. Run transformation

Benefits

  • Reduces manual data entry
  • Improves data accuracy
  • Efficient relationship mapping
  • Scalable for large datasets
